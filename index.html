📋 Cara Import .ics ke Google Calendar

  1. Download file .ics dari tombol di atas
  2. Buka Google Calendar di browser
  3. Klik ikon ⚙️ Settings (gear) di kanan atas
  4. Pilih Import & Export dari menu kiri
  5. Klik "Select file from your computer"
  6. Pilih file .ics yang sudah di-download
  7. Pilih kalender tujuan, lalu klik "Import"
  8. Semua task akan muncul di Google Calendar!
